Math Lesson

Addition

This lesson will teach you how to add numbers.

  1. Step 1: Find the first number.
  2. Step 2: Find the second number.
  3. Step 3: Add the two numbers together.

For example, to add 2 and 3, you would first find the first number, which is 2. Then, you would find the second number, which is 3. Finally, you would add the two numbers together, which is 5.

Here is a practice problem:

What is 5 + 4?
